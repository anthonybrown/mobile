Geo Tweet

My first app with sencha touch !

Welcome to the mobile site for my mobile class at NEiA, I will be testing out Sencha Touch to build a geo tweet application with the Google map api. I am also using media queries and jQuery to help

The GeoTweet application is using the Twitter API  I have it set to a 50 mile radius of where the device that is sending the query. I have not been able to figure out how to do much else with it yet, like a back button (the refresh button in the top right does not work.So far warning,  you have to use the back button to navigate!
